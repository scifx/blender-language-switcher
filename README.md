# blender-language-switcher
## 概述
- 这是blender的界面语言切换插件
- 一键切换界面的中英文显示
- 方便看教程学习，及调试语言导致的相关错误

## 用法提要
- 在blender插件面板勾选启用插件后
- 直接点击菜单栏的“语言”，点击即可来回切换中英文界面显示
- 快捷键"Ctrl+Shift+L"
